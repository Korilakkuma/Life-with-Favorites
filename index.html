Career

大学時代の専攻は, 気がついたら興味はないのに建築. 大学末期に, FORTRAN のプログラミング講義を受講したことをきっかけに, コンピュータの仕組みに興味をもち, 基本情報技術者を取得. また, 初歩的な C 言語のコードなら実装できるレベルになる.

専攻を情報科学に変更して大学院に進学. 音情報処理を研究するも, うつになる. 某インターネット企業 (Y 社) から内定をもらうが, うつの治りがよくなく, 入社辞退.

3 年ほどニート生活. この間に, コンピュータサイエンス, Web, プログラミング (特に, JavaScript), ソフトウェア設計などを学び, 職業プログラマとしての基礎中の基礎を独学. 2014 年に, 運よく, 某インターネット企業 (D 社) に内定をもらい, 今度は無事入社するも, 1 年半ほどで, 某インターネット企業 (C 社) に転職.

配属された事業のメンバーや Web フロントエンドエンジニアに恵まれて, ソフトウェアエンジニア, Web フロントエンドエンジニアとして一定の市場価値を得られるまでになる.
